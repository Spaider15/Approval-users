Для функционирования приложения требуется создать базу данных с названием myapp.
Далее, импортировать базу myapp.sql, находится в корне проекта.
В файле app.js с 9 по 14 строчку, изменить настройки подключения к базе.
В корне выполняем set DEBUG=myapp:* & npm start.
POST запрос с JSON отправляем на URL: http://localhost:3000/users

Формат JSON в файле: lesson_ajax.users.json
